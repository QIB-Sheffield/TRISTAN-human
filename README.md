Data structure
--------------

The data should be organised as follows:


MAIN DIRECTORY
  |
  |header.csv
  | 
  |---SubjectID1
  |	|
  |	|--Visit1Scan1
  |	|--Visit1Scan2
  |	|--Visit2Scan1
  |	|--Visit2Scan2
  |---SubjectID2
  |	|
  |	|--Visit1Scan1
  |	|--Visit1Scan2
  |	|--Visit2Scan1
  |	|--Visit2Scan2
  .	.
  .	.
  .	.

Detail the Subject IDs and Scan directory names in your data_finder.csv for 
navigation within the main directory path.
